## Installing :hammer_and_wrench:

# Env

As in the xayn_search app, create `.env.dev` and `.end.pod` in the root folder.
Use `.env.example` as a template.

after that, run the following bash script:

```shell
$ flutter pub get
$ cd application
$ flutter pub run build_runner build --delete-conflicting-outputs
$ cd -
```
